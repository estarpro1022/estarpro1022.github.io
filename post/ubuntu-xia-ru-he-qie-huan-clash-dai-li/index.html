ubuntu下如何切换clash代理

2023-10-09 1 min read # linux # clash

我们代理将要到期的时候，可能会更改另一家代理，那么原来的代理就作废了。

Windows

Windows下只需要在clash的订阅里手动切换一下即可

Ubuntu

Ubuntu下用命令行会操控clash稍微麻烦点。将Windows下代理的内容粘贴到 /etc/clash/config.yaml 和~/.config/clash/config.yaml ，然后 sudo systemctl restart clash 即可。用curl www.google.com验证代理是否可用，如果输出一段html代码则证明可用

我们再看一下Windows下的代理内容，右键上图中选中的代理，然后点击 Edit，弹出窗口的内容就是我们所需要的啦。

这里再推荐一个网页版的clash管理面板，可以切换不同节点，在linux下管理会相对方便一些

  • 秘钥可以到clash的配置文件中看，Windows是~/.config/clash/config.yaml，里面包含端口号和秘钥
